# ETHConf x ETHGlobal - Live Tweet Wall

Pinterest-style wall of LIVE embedded tweets.

- Paste a tweet link -> embeds + auto-tags by event (#ethconf2026, #newyork2026, #lisbon2026, #cannes2026, #general; year auto-detected)
- Date captured from each tweet's ID (Snowflake) -> sort Oldest/Newest
- Filter tabs by tag; added links saved in your browser
- 42 seed tweets (23 ETHConf, 19 ETHGlobal New York)

Static site (no build). Deploys on Vercel automatically on every push to this repo.
